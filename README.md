# VIT Vellore Smart Shuttle System: An AI-Powered Campus Commute Optimization Platform

An interactive shuttle booking and tracking platform designed to optimize campus transportation using data science and machine learning. The system provides real-time shuttle visibility, predictive arrival times, and a seamless booking experience for students.

---

## Overview

The VIT Vellore Smart Shuttle System is built to address inefficiencies in campus transportation such as unpredictable wait times, lack of real-time tracking, and absence of booking mechanisms.

The platform integrates machine learning, real-time data visualization, and backend services to deliver a smart and efficient commute experience within a large university campus.

---

## Problem Statement

Campus transportation systems often suffer from:

- Lack of real-time shuttle tracking  
- Unpredictable wait times  
- No advance booking system  
- Inefficient scheduling and congestion during peak hours  

This project addresses these challenges by combining predictive analytics, real-time tracking, and user-centric design.

---

## Objectives

- Provide real-time shuttle tracking across campus  
- Predict shuttle arrival times using machine learning  
- Enable secure shuttle booking for students  
- Deliver personalized commute assistance  
- Visualize transportation data through analytics dashboards

## Key Features

1. Real-Time Tracking  
   Interactive campus map built using Leaflet.js to display live shuttle and stop locations  

2. Personal AI Commute Assistant  
   Provides personalized shuttle reminders and updates upon user login  

3. Machine Learning-Based Predictions  
   Custom-trained models forecast shuttle arrival times using simulated historical data  

4. Firebase-Based Booking System  
   Secure login using phone authentication and real-time booking storage using Firestore  

5. Interactive Analytics Dashboard  
   Displays ridership trends and popular stops using dynamic charts  

---

## System Architecture

1. Data Simulation and Processing  
   - Generated and processed shuttle movement data using Python  

2. Machine Learning Layer  
   - Built predictive models to estimate shuttle arrival times  

3. Backend Integration  
   - Python with Eel for communication between frontend and backend  

4. Frontend Interface  
   - Developed using HTML, Tailwind CSS, and JavaScript  

5. Real-Time Services  
   - Firebase Authentication for login  
   - Firestore for booking data storage  

6. Visualization  
   - Leaflet.js for real-time mapping  
   - SVG and JavaScript charts for analytics
